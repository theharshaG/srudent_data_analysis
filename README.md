# srudent_data_analysis
Student Data Analysis (Python)

##  Project Description
This is a simple Python program that performs analysis on student marks using a dictionary.
The program demonstrates basic Python concepts like:
- Dictionary
- Loops
- Conditional statements
- Sorting
- Lambda functions
- Dictionary comprehension

## Student Data
The program uses the following student marks:
Harsha → 85  
Ravi → 72  
Anu → 90  
Kiran → 60  

##  Features

### 1. Students Scoring Above 75
Prints the names and marks of students who scored more than 75.

### 2️.Maximum Marks
Finds the highest marks manually using a loop.

### 3️. Class Topper
Identifies the student with the highest marks using the `max()` function.

### 4️.Top 3 Students
Sorts students based on marks in descending order and displays the top three.

### 5️. Pass / Fail Result
Creates a dictionary showing:
- "Pass" if marks ≥ 75  
- "Fail" if marks < 75  

## Concepts Used
- Python Dictionary
- for loop
- if condition
- max() function
- lambda function
- sorted() function
- list slicing
- dictionary comprehension
